# v1.0.0

Дата релиза: `2026-03-26`

## Кому особенно важно прочитать этот релиз

- тем, кому нужен первый стабильный публичный baseline проекта
- тем, кто хочет понять, какие core surfaces впервые стали нормальным production contract
- тем, кто сравнивает более новый релиз с исходным stable launch

## Что зафиксировал этот релиз

- первый стабильный публичный контракт для core CLI
- утверждённый стабильный набор Claude и Codex events
- первый стабильный путь интеграции через Go SDK
- generated scaffold rules для поддерживаемых stable paths

## Как лучше читать этот релиз

- Владельцу нового repo: как baseline note, который показывает, что именно впервые стало stable, а не как единственную актуальную рекомендацию на сегодня.
- Владельцу существующего repo: как способ понять, какие promises уже были stable до более поздних изменений guidance.
- SDK- или platform-consumer: как более широкий stable floor, который потом нужно дополнить более узкими релизами по своей поверхности.

## Что было stable в этом релизе

- CLI: `init`, `validate`, `capabilities`, `install`, `version`
- Claude stable event set: `Stop`, `PreToolUse`, `UserPromptSubmit`
- Codex stable event set: `Notify`
- root Go SDK integration path

## Что осталось beta

- optional extras из `plugin-kit-ai init --extras`
- любая будущая область, которую ещё не перевели в stable contract явно

## Чего этот релиз не означал

- Он не означал, что каждый будущий target или helper path автоматически получил такой же stable promise.
- Он не заморозил будущую delivery guidance для Python или Node.
- Но именно он зафиксировал первый действительно надёжный публичный floor для проекта.

## Важная migration note

SDK registration переехала с root-Claude methods на platform registrars. Если вы строились против pre-stable shapes, проверьте SDK wiring до того, как использовать `v1.0.0` как compatibility baseline.
